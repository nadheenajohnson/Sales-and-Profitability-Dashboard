# Sales & Profitability Dashboard

## Project Overview
This Power BI dashboard analyzes sales, profit, orders, and profitability. The dashboard helps identify top-performing categories, profitable regions, and business trends to support data-driven decision-making.

## Objectives
- Analyze overall sales performance
- Monitor profitability across categories
- Identify top and bottom-performing regions
- Track order trends and business growth
- Provide actionable business insights

## Tools Used
- Power BI
- Microsoft Excel

## Key KPIs
- Total Sales
- Total Profit
- Total Orders
- Profit Margin %

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Profit Analysis by Category
- Regional Performance Analysis
- Interactive Filters and Slicers

## Key Insights
- Technology category generated the highest sales.
- Profitability varies across categories and regions.
- Some regions contribute significantly more profit than others.
